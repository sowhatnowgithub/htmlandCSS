overflow attribute is used to control the content

Changing Inline/block

display property allows you to turn an inline element to block or vice versa, by setting display:inline will make the elements to turn a block into inline

display:block will turn an inline to block

display:none will not show in the page, but can be viewed in the source page

display:inline-block this will make a block-level element to flow like an inline element and retaing the other features of block-level elemets
