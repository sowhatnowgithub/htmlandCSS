Forms

form controls, these are responsible for the input of user, like what type of data, they are taking ex: integers, date, radio buttons, drop down, choices etc..

To differentiate the inputs given by the user, the html stores the input of user as name-value pairs for use and for the transfer of data

You should never change the name of a form control in a page unless you know that the code on the server will understand this new value.

Attributes of form element

  1. action: This contains the URL to which the server-side request should be sent to
  2. method: This contains what type of request is supposed to be done, which takes get or post, default value is get if not specified
  3. input: This is an empty element which takes input and inline element of form tag
  4. type: Attribute of input tag, which says which type of data will the input be
  5. name: Attribute of input tag, which is the identity of the input
  6. size: attribute of input tag, this is old and it gives the input size
  7. maxlength: attribute of input tag, this tells us the maxsize of the input data
  8. textarea: this is an inline element for the form tag, which creates a text area
