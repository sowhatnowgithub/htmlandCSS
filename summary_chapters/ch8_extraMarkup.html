meta

This lives in the head of the html code, it is a empty element

  1. name and content are the important attributes of the meta tag, these two goes together
    • description: value of attribute name, this means the attribute content will be about description of the page
    • keywords: value of attribute name
    • robots: value of attribute name, content can take value noindex which means the page cannot be added to search engine and nofollow value implies it can be added to search engine but cannot add the pages it links to
  2. http-equiv is another attribute that is used along with content
    • author is the value taken by attribute http-equiv, then content takes the author name
    • pragma is the value taken by attribute http-equiv, the value no-cache tells the browser not to canche it locally
    • expires is the value taken by attribute http-equiv, this value tells the browser when the page will expires

  3. The value of the name attribute is the property you are setting, and the value of the content attribute is the value that you want to give to this property.
