Table

  • table: this is a main tag to intiate a table
  • tr: table row this is like ol(ordered list) this intiates the row
  • td: table data this is like li this intiates the column value
  • th: table heading, this has a attribute called scope which takes value col and row
  • scope This is an attribute used to define the heading of the column or a row, and it takes the value row or col

Long Tables

  1. thead is the inline element for the table tag, which has all the column headings
  2. tbody is the inline element for the table tag, which has all the content
  3. tfoot is the inline element for the table tag, which can final outputs

Old attributes before html5

These attributes are usually placed in table tag but not nessarily

  1. width is a attribute controls the size of the cell
  2. cellpadding is an attribute that does padding of the content
  3. cellspacing is an attribute that does the space between the cell
  4. border this gives a border of said value
  5. bgcolor this sets the color of the cell
