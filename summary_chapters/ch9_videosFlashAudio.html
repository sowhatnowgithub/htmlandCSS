This video tag introduced in the html5 is widely used, the video format have to supported by the browser

The usage of the video can be done in two ways

  1. Using Hosted Video Services: This method is used to use platforms liek youtube or vimeo
    to link the video to website, this has disadvantages and advantages, Check them online
  2. Locally uploading the video ,and then loading it, this method takes time when have to load via server-side
  • video: this is the tag used for the video
  • src: This attribute provides either the absolute or relative URL for the filepath
  • poster: This attribute allows for the image to be displayed while the video loads, this attributes takes the URL for the image filepath
  • width, height
  • controls: this attribute takes no value, just by placing them in the tag is on, if not placed then off
  • autoplay: This attribute takes no value, same like controls
  • loop: This attribute takes no value, this means the video plays after the video ends in loop
  • preload: This attribute tells the browser what to do when the page loads, it can have three values
    1. none: This value tells, the browser should not load the video until the user presses play
    2. auto: This value tells, the brower should download the video when the page loads
    3. metadata: This value tells, the browser should just collect information such as the size, first frame, tract list and duration
  • source: This inline tag replaces the src attribute of the video tag and it has the attribute src which takes the value of the filepath,
    This source tag is useful, as we can place multiple video formats in differt source inline tags, Ex:screenshot
    1. type: This is an attribute of the tag source which tells the borwser what video format is the video, otherwise the browser will download some video and checks whether it can play or not

      2. Ex:screenshot

    2. codecs: This codecs used within the type attribute of the source tag ,The codec that was used to encode the video is supplied within the type attribute. Note the use of single quotes, as well as double quotes in the type attribute, when it is supplied.
