Rules for images

1.Choosing Right image format
  1. jpeg : If the image has many colors
  2. png or gif : If the image is flat image i.e, if the image has only few dominant colors usually for logos
  3. jpg's, gif's, png's these are example of bitmap which means these are made up of a lot of miniature squares
  4. Vector Images: These are completely different from bitmap images, as these are formed by joining lines between two points
    • The advantage of drawing line images we can change resolution without affecting quality
  5. SVG: Scalable Vector Graphics is a new image format
2.Choosing Right size
  1. As wrong size would take lot space and rendering will result in distortion and such
  2. ITs better to save image with the same resolution as the you would need it in the website as it would save time
3.Choosing right resolution
  1. using right resolution will save time in loading of the images
  2. Resolution of an image is the number of squares that fit within 1 inch * 1 inch size
  3. ppi: pixels per inch, 72 ppi are usually good resolution for an image
