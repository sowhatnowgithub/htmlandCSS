BULLET POINT STYLES

list-style-type: With this property we can change the style of bullet point also known as marker

Unordered Lists

  • list-style-type: none - this will remove the marker
  • list-style-type: disc - this will set it to dark circle
  • list-style-type: circle - border dark and white inside
  • list-style-type: square - this will set it to square

Ordered Lists

  1. list-style-type: decimal
  2. list-style-type: decimal-leading-zero
  3. list-style-type: lower-alpha - this stands lower alphabet
  4. list-style-type: upper-alpha
  5. list-style-type: lower-roman
  6. list-style-type: upper-roman

Border on empty cells

empty-cells

  1. empty-cells: show - this shows border of empty cells
  2. empty-cells: hide - this hides the the border
  3. empty-cells: inherit - if it is nested table, this tells to obey the rules of containing table
